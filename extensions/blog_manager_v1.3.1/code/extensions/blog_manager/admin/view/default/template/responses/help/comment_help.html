Comments

Comments are sorted by most recent article release date first then most recent comment date. You can filter comments for a given article by selecting the article from the drop down.

Listing Grid View

Each comment thread is grouped with corresponding replies. The amount of replies for a given comment are indicated. If you are moderating comments, the amount of unapproved comments are also indicated. You can change the comment status and the approval status from the listing grid or by editing the comment or reply. You can also directly edit a comment or reply, reply to a comment or delete a comment.

Edit Comments

Along with approval actions, you have full edit ability for each comment/reply. When in edit mode, changes you make and save will be instantly live on your blog. You also have the option of sending out notifications of your changes to any users who have requested notifications. Notifications should be sent with comment approval but you may not want to send notifications if you editied the comment/reply.

Reply to Comments

You can directly reply to comments by clicking the reply icon. You have the option of replying as the author of the article, the blog owner, or as a blog admin, provided you have created a user with admin permissions. Your comments will be live once you save them. You also have the option of sening out notifications to subscribers.

In both edit and reply mode, a link is provided to open a window showing the article content and the parent comment as a reference.
